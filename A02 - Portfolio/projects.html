SECJ2013
Data Structures and Algorithms

Browser history management system

The project is a browser history management system that uses the Queue data structure based on the First-In-First-Out (FIFO) principle. It consists of two main classes: Queue, which handles operations like enqueue, dequeue, and peek, and History, which manages the browsing history. The system allows users to navigate backward and forward in their browser history, providing a seamless browsing experience. The FIFO concept ensures that the history is maintained in the correct sequence for navigation purposes.

SECR1033
Computer Organization and Architecture

CPU & Hardware Performance Benchmarks

The project focuses on evaluating CPU and hardware performance benchmarks across different laptops. The aim is to understand CPU benchmarking principles and evaluate how various factors such as workloads, memory usage, and hardware specifications impact performance. Through synthetic and system-level benchmarks, the team compared processors by running tests using tools like Geekbench and CPU-Z. The findings reveal significant performance differences, emphasizing how generational CPU changes influence execution speed and data access.
